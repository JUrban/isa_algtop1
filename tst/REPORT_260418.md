## 3. Architecture

### 3.1 Session Structure

The project is organized into two Isabelle sessions:

- **`Top0`** (cached, `tst/i/`): General topology, Chapters 2–8. Four theory files totaling 87,066 lines with 1,020 results and only 11 sorries. This provides the foundation: topological spaces, continuous maps, compactness, connectedness, product/quotient topologies, and metrization theorems.

- **`AlgTop`** (`tst/`): Algebraic topology, Chapters 9–14. Single theory file `AlgTop.thy` importing `Top0` and Isabelle's `Complex_Main`.

### 3.2 Topology Framework

The formalization uses a custom topology framework (`top1_*` predicates) rather than Isabelle's built-in `topological_space` class. This design choice, inherited from the base session, represents topologies explicitly as sets of open sets:

```isabelle
definition is_topology_on :: "'a set ⇒ 'a set set ⇒ bool"
definition top1_continuous_map_on :: "'a set ⇒ 'a set set ⇒ 'b set ⇒ 'b set set ⇒ ('a ⇒ 'b) ⇒ bool"
```

This approach enables working with multiple topologies on the same carrier (essential for subspace, product, and quotient topologies) but requires explicit bridge lemmas to connect with Isabelle's standard `continuous_on` and `open` predicates.

### 3.3 Key Bridge Lemmas

A significant engineering effort went into building bridges between the custom topology framework and Isabelle's standard topology:

- **`top1_continuous_map_on_subspace_open_sets`** (polymorphic): Transfers `continuous_on UNIV f` to `top1_continuous_map_on` with subspace topologies. Works for any type with `open` (real, complex, products).
- **`top1_continuous_map_on_real2_subspace_general`**: Specialized R² version for the deformation retract proofs.
- **`II_topology_eq_subspace`**: The I×I product topology equals the subspace of R² topology.
- **`product_topology_on_open_sets`** (polymorphic): Product of standard topologies equals standard product topology.

## 4. Major Proved Results

### 4.1 Theorem 58.7: Homotopy Equivalence Induces π₁ Isomorphism

**Statement:** If `f: X → Y` is a homotopy equivalence with homotopy inverse `g`, then `f_*: π₁(X, x₀) → π₁(Y, f(x₀))` is a group isomorphism.

This is one of the central theorems in algebraic topology. The proof required developing substantial infrastructure:

**Injectivity** was proved using the basepoint-change argument: from g∘f ≃ id, derive that g_* ∘ f_* equals a basepoint-change isomorphism, hence f_* is injective.

**Surjectivity** was the harder direction, requiring:
1. **`homotopy_induced_basepoint_change`** (Lemma 58.4): If H is a homotopy from h to k and l is a loop at x₀, then h∘l is loop-equivalent to a basepoint change of k∘l. This proof needed:
   - I×I is simply connected (straight-line contraction homotopy)
   - Convexity lemmas for affine combinations in [0,1]
   - 5-step path algebra transitivity chain
   - `continuous_on` bridge via `continuous_on_open_invariant`

2. **`double_basepoint_change_equiv`**: bc(α, bc(β, f)) ≃ bc(β·α, f). Proved via:
   - `rev(β·α) = rev(α)·rev(β)` (extensional equality with `field_simps`)
   - Three applications of `Theorem_51_2_associativity`
   - `path_homotopic_product_right` for lifting inner homotopies

3. **Explicit preimage construction**: For any loop m at f(x₀), take m' = bc(rev(γ), m) and show f_*([bc(α₁, g∘m')]) = [m] via the computation chain:
   f∘bc(α₁, g∘m') = bc(f∘α₁, f∘g∘m') ≃ bc(f∘α₁, bc(rev α₂, m')) ≃ bc(γ, m') ≃ m.

The full surjectivity proof is approximately 156 lines of structured Isar.

### 4.2 Fundamental Theorem of Algebra (Steps 3–4)

**Statement:** Every non-constant complex polynomial has a root.

The FTA proof follows Munkres' topological argument in four steps. Steps 3 and 4 are now fully proved (0 sorries):

**Step 3** (the core): For a polynomial k(z) = z^n + Σ(aⱼzʲ) with Σ|aⱼ| < 1, k has a root in the closed unit disk. Proved by contradiction:
- k nonvanishing on B² implies h = k|_{S¹} extends to B², hence h is nulhomotopic
- The straight-line homotopy F(z,t) = z^n + t·Σ(aⱼzʲ) connects z^n to h in C−{0}
- By nulhomotopy transitivity, z^n would be nulhomotopic — contradicting Step 2

Key subproofs:
- **h nulhomotopic**: Homotopy H(z,t) = k((1−t)z) via norm computation `|1−t|·|z| ≤ 1` using `norm_of_real` and `of_real_diff`
- **F continuous**: Product topology transfer via `Theorem_16_3` + `product_topology_on_open_sets` for the mixed complex×real type
- **z^n continuous** (S¹→S¹ and S¹→C−{0}): Via the polymorphic `top1_continuous_map_on_subspace_open_sets` helper

**Step 4**: Reduction of the general case to Step 3 by substitution x = cy for large c. Already proved.

The FTA chain needs only **3 more sorries** (in Steps 1–2), all requiring π₁(S¹) ≅ ℤ.

## 5. Proof Engineering Techniques

### 5.1 The Sorry-First Workflow

All proofs follow the discipline mandated by `CLAUDE.md`:
1. Write the full proof structure using only `sorry`
2. Compile immediately to verify the structure
3. Replace sorries in small batches using `sledgehammer` / `process_theories`
4. Use `by (by100 tactic)` wrappers limiting each tactic to 100ms

This discipline prevents session-killing tactic explosions and ensures every proof step is locally attributable.

### 5.2 Topology Transfer Patterns

The most common proof pattern in the formalization is the **topology transfer**: proving a continuity fact in the standard topology (`continuous_on`) and transferring to the custom topology (`top1_continuous_map_on`). The polymorphic helper `top1_continuous_map_on_subspace_open_sets` reduced many 30-line proofs to 3-line applications:

```isabelle
lemma top1_continuous_map_on_subspace_open_sets:
  assumes "⋀x. x ∈ S ⟹ f x ∈ T" and "continuous_on UNIV f"
  shows "top1_continuous_map_on S (subspace_topology UNIV top1_open_sets S)
                               T (subspace_topology UNIV top1_open_sets T) f"
```

### 5.3 Path Algebra Infrastructure

The path algebra (§51) forms the foundation for all subsequent proofs. Key lemmas:
- Associativity, left/right identity, left/right inverse (all fully proved)
- `path_homotopic_product_left/right`: Composition preserves homotopy
- `double_basepoint_change_equiv`: Crucial for Theorem 58.7 surjectivity
- `top1_path_reverse_twice`: rev(rev f) = f (definitional equality)
- `rev(β·α) = rev(α)·rev(β)` (extensional equality)
