```
  "π₁(C) ≅ ℤ ∧ π₁(X) ≅ ℤ ∧ j_* hom ∧ j_* nontrivial ⟹ j_* iso"
```

More precisely: if G ≅ ℤ and H ≅ ℤ and f: G → H is a nontrivial homomorphism,
then f is an isomorphism.

Proof: f maps a generator g of G to f(g) ≠ eH. In ℤ, any nonzero element
generates a subgroup nℤ. If f is a hom and f(g) = n ≠ 0, then f is injective
(ker f = {0} since g generates G and f(g) ≠ 0). And |im f| = |G/ker f| = |G| = ∞,
so im f = nℤ. For surjectivity we need n = ±1.

Actually, the textbook argument is simpler and does NOT need this general fact.
It uses:

1. α*β is a generator of π₁(X) (from Theorem 63.1 + infinite cyclic)
2. α*β ∈ image(j_*) (since α*β lies in C)
3. Therefore j_* is surjective
4. Surjective hom ℤ → ℤ ⟹ iso (since ker ⊆ ℤ, ℤ/ker ≅ ℤ, only ker = {0} works)

So we need: **α*β is a generator**, not just nontrivial.

But ALSO: π₁(C) ≅ ℤ, so image(j_*) ≅ ℤ/ker(j_*) is a quotient of ℤ.
A quotient of ℤ that's infinite must be ℤ itself (ker = 0).

## Simplest Implementation

Since both π₁(C) and π₁(X) are ≅ ℤ, and j_* is a nontrivial hom between them,
it suffices to prove: **nontrivial hom ℤ → ℤ is an isomorphism**.

This avoids the generator question entirely. A hom ℤ → ℤ maps 1 ↦ n. Nontrivial
means n ≠ 0. Then k ↦ kn is injective. Image = nℤ ≅ ℤ. But nℤ = ℤ iff n = ±1.
For n = ±1: bijective ✓. For |n| > 1: image is a proper subgroup, contradiction
with π₁(C)/ker ≅ im ≤ ℤ and π₁(C) ≅ ℤ... actually nℤ ≅ ℤ for all n ≥ 1, so
this doesn't give a contradiction!

The issue: a hom ℤ → ℤ mapping 1 ↦ 2 IS injective with image 2ℤ ≅ ℤ. So
"nontrivial hom ℤ → ℤ" does NOT imply "isomorphism." It's injective but
not necessarily surjective.

So we DO need to show α*β is a GENERATOR (n = ±1), not just nontrivial.
The textbook is right that this requires the full strength of Theorem 63.1.

## CORRECT ARGUMENT (following textbook)

α*β is a generator because of Theorem_63_1_c_subgroups_trivial:

1. [α*β] = gen^n for some n ≠ 0 (nontrivial + π₁(X) infinite cyclic)
2. Suppose |n| > 1. Then gen = [α*β]^{1/n}... but we can't take fractional powers.
3. Instead: gen is a loop in X = U ∪ V. As a loop, it can be decomposed as γ*δ
   where γ is a path in U and δ is a path in V (by subdivision of the path).
4. Then (α*β)^1 = gen^n = (γ*δ)^n as homotopy classes in π₁(X).
5. By Theorem_63_1_c_subgroups_trivial: (α*β)^m ≃ (γ*δ)^k ⟹ m = 0.
6. But m = 1 ≠ 0. Contradiction. Hence n = ±1 and α*β is a generator.

Step 3 is the subtle part: showing gen decomposes as γ*δ with γ in U, δ in V,
and the endpoints in the correct components of U∩V.
